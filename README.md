# Pinheiro - Currency Converter

A simple, fast, and responsive currency converter built with **HTML, CSS, and vanilla JavaScript**, using a real-time exchange rate API.

---

## Features

- Convert between:
  - 🇧🇷 Brazilian Real (BRL)
  - 🇺🇸 US Dollar (USD)
  - 🇪🇺 Euro (EUR)
- daily exchange rates via API
- Automatic currency formatting
- Clean and responsive UI
- Dynamic flag updates based on selected currencies

---

## API

This project uses https://hexarate.paikama.co/ 
It returns the latest exchange rate between two currencies.
